# Visualizing Zipf's law in R

This R script visualizes Zipf's law using a word frequency list.

## Data

Word counts file from http://norvig.com/ngrams/.

The format of the data file is as follows:

    WORD	FREQ
    the   1000000
    a      999999
    ...
